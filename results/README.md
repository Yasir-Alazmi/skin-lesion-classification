# Experiment Results

This directory contains all outputs from the skin lesion classification experiments on HAM10000.

---

## Visualisations

| File | Description |
|------|-------------|
| [`class_distribution.png`](class_distribution.png) | Bar chart showing HAM10000 class imbalance (nv dominates at 66.9%) |
| [`training_history.png`](training_history.png) | Loss & accuracy curves for EfficientNet-B3 and ViT-B/16 across epochs |
| [`confusion_matrices.png`](confusion_matrices.png) | Normalised confusion matrices for both models on the test set |
| [`roc_curves.png`](roc_curves.png) | Per-class One-vs-Rest ROC curves with AUC scores |
| [`sensitivity_specificity.png`](sensitivity_specificity.png) | Per-class sensitivity (recall) and specificity comparison |
| [`model_comparison.png`](model_comparison.png) | Side-by-side bar chart of Accuracy, F1, and AUC for both models |
| [`hair_removal_demo.png`](hair_removal_demo.png) | Grid showing original vs. hair-removed images (Black-Hat + TELEA) |
| [`EfficientNet-B3_predictions.png`](EfficientNet-B3_predictions.png) | Sample test images with EfficientNet-B3 predictions and confidence |
| [`ViT-B_16_predictions.png`](ViT-B_16_predictions.png) | Sample test images with ViT-B/16 predictions and confidence |
